# types-affine

Types for the Affine package

## Install

```
pip install types-affine
```

These type definitions should automatically be picked up by Mypy when type
checking.

## Versions

The versions of affine that `types-affine` tracks.

| types-affine | affine |
|:-------------|:-------|
| 0.1.x        | 2.3.x  |
